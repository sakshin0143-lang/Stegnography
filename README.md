This project implements image steganography using the Least Significant Bit (LSB) technique in C. It allows hiding and extracting secret data within BMP image files.

Features :
1.Encode secret text data into a BMP image
2.Decode hidden data from an encoded image
3.Uses Least Significant Bit (LSB) manipulation
4.Modular implementation with separate encoding and decoding logic
5.Supports basic file handling for secret data
6.Lightweight and efficient C program
7.Works with standard 24-bit BMP images
